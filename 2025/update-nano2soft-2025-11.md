## 2025-11-19 – 2025-11-21 Support Swagger OpenApi
تم دعم ال Swagger / OpenAPI فى جزء الapi الخاص بنا.

**ما هو Swagger / OpenAPI؟**

باختصار، OpenAPI هو مواصفة قياسية (واجهة) لوصف وثائق واجهات برمجة التطبيقات (APIs). أما Swagger فهو مجموعة من الأدوات مفتوحة المصدر (مثل Swagger UI) التي تُنفذ هذه المواصفة. بمعنى آخر، OpenAPI هو "النظرية" و Swagger هو "التطبيق العملي".

**أبرز المزايا:**

1. توثيق شبه تلقائي وحيوي للـ API

2. واجهة مستخدم تفاعلية (Swagger UI)

بدلاً من قراءة مستند نصي ثابت، يصبح لدينا الآن واجهة ويب غنية بقدرات هائلة:
- تصفح جميع نقاط النهاية (Endpoints): يمكنك رؤية جميع GET, POST, PUT, DELETE requests وغيرها في مكان واحد.
- اختبار الـ API مباشرة من المتصفح: يمكنك النقر على أي endpoint وتجربته مباشرة من خلال الواجهة. يمكنك إدخال البيانات المطلوبة والنقر على "Execute" وسيتم إرسال الطلب فعليًا إلى الخادم وعرض النتيجة.
- رؤية هياكل البيانات المتوقعة: تُعرض هياكل JSON المتوقعة للإرسال (Request Body) والاستقبال (Response Body) بشكل واضح، بما في ذلك أنواع البيانات (String, Number, Boolean).

3. تبسيط تطوير الواجهة الأمامية (Frontend)

لمطوري الواجهة الأمامية، هذه الميزة كنز:
- لم يعودوا بحاجة إلى انتظار المطور الخلفي (Backend) ليكتب وثيقة أو يخبرهم بشكل البيانات.
- يمكنهم رؤية جميع الـ endpoints المتاحة وتجربتها بأنفسهم لفهم كيفية عمل النظام.
- يمكنهم استخدام هياكل الـ Response لبناء واجهة المستخدم بدقة وسرعة.

4. تحسين التواصل وتبادل المعلومات

- داخل الفريق: يصبح التواصل بين أعضاء الفريق (مطورين خلفيين، واجهة أمامية، مديري مشاريع، فاحصي جودة) أكثر سلاسة وفعالية.
- مع العالم الخارجي: إذا كان لدينا واجهات برمجة تطبيقات عامة (Public APIs) أو نتعاون مع فرق خارجية، فإن وثائق Swagger/OpenAPI هي المعيار العالمي لتقديم وشرح قدرات نظامنا.

**الخلاصة**

إضافة دعم Swagger / OpenAPI هو أكثر من مجرد "تحديث تقني". إنه نقلة نوعية في:
- الجودة: وثائق دائمة الدقة والحداثة.
- الكفاءة: توفير هائل في وقت التطوير والاختبار والتواصل.
- الاحترافية: اتباع أفضل المماريات والمعايير العالمية في عالم تطوير الـ APIs.

هذه الميزة ستجعل العمل على المشروع أسرع، أوثق، وأكثر متعة للجميع.

```
لتحميل ملف collection api بصيغة swagger OpenApi v 3.0.0

https://account.now-ye.com/api/v1/thunder/collection/api?is_open_api=1

للاطلاع على واجهات التوثيق بصيغة swagger-ui

https://account.now-ye.com/docs/swagger?url=https://account.now-ye.com/api/v1/thunder/collection/api?is_open_api=1
```
