#Update 2025-3

**تحديثات شهر ثلاثة **

## 2025-01-01 إلى 2025-03-02
**SEO Management Version 2**

إطلاق الوحدة البرمجية المتخصصة في إدارة السيو وتحسين ظهور الموقع على محركات البحث حيث تم برمجة وحدة برمجية متكاملة متخصصة بهذا الموضوع تشمل كافة جوانب السيو على مستوى كافة الصفحات والكائنات في النظام. تهتم هذه الوحدة بالتالي:

1. `meta title` و`site name` و`description` و`keywords`
2. دعم الوسائط الاجتماعية (Facebook وTwitter)
3. دعم توليد `schema.org` JS
4. دعم `extra meta`
5. دعم `sitemap` (index, page, images, videos)
6. دعم `robots`
7. دعم `humans.txt`
8. دعم `favicon` بأحجام مختلفة
9. دعم توليد `webmanifest` (frontend وbackend)

* تم تطوير هذه الوحدة بشكل احترافي بعد الأخذ بعين الاعتبار كافة جوانب السيو وتم اختيار أفضل الممارسات عند برمجة وتصميم هذه الوحدة وتعتبر خلاصة لكافة البرمجيات السابقة المتعلقة بموضوع السيو وتحسين محركات البحث حيث تم الاستفادة من خبراتنا السابقة وعكسها بشكل مثالي في هذه الوحدة.
* في المرحلة الأولى سيتم تفعيل هذه الوحدة على مستوى المواقع الديناميكية الغير معتمدة على الـ API.
* في المرحلة الثانية سيتم تفعيل الوحدة في جزء الـ API على مستوى الصفحات الديناميكية وعلى مستوى الكائنات الأخرى كصفحات الأصناف والأخبار والمقالات وغيرها.

```plaintext
POST /accounts/students/balances
```

```plaintext
POST /accounts/students/details
```

```plaintext
GET /accounts/students/transactions
```

```plaintext
POST /accounts/students/payment
```

**للتحقق من وجود حركة مالية معينة يمكن استخدام الرابط التالي مع تمرير رقم الحركة:**

```plaintext
POST /api/v1/accounts/transactions/check
```

هذه الخطوة تمكننا من التعامل مع موصلين يعملون في ساعات محددة وفي أيام محددة فقط.

**ملاحظة:** لتفعيل التعامل مع أوقات العمل على مستوى كل موصل على حدة يجب أن يتم تفعيل الخيار التالي لكي تظهر واجهات إدارة أوقات العمل ضمن جزء الموصلين، علماً أن القيمة الافتراضية لهذا الخيار هي عدم تفعيل (`false`):

```plaintext
# السماح بالتعامل مع أوقات العمل على مستوى كل موصل على حدة 
NANO_DELIVERYS_DELIVERY_ALLOW_WORKING_HOURS = true
```

يمكن مشاهدة الفيديو والصور للتوضيح.

```plaintext
NANO_WORKING_DEPARTMENT_ALLOW_FILTER_WORKING_HOURS = true

# السماح بإضافة الفلاتر الخاصة بأوقات العمل إلى واجهة الموصلين 
NANO_DELIVERYS_DELIVERY_ALLOW_FILTER_WORKING_HOURS = true
```

يمكن مشاهدة الفيديو والصور للتوضيح.
