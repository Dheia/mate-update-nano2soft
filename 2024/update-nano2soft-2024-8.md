#Update 2024-8 

**تحديثات شهر  ثمانية**


## من أواخر شهر 7-2024 إلى 2024-08-01
**برمجة مكتبة احترافية خاصة بتفقيط الأرقام**

أي تحويل الرقم من رقم إلى كتابة بحيث تدعم المكتبة تحويل الأرقام إلى أكثر من عشرين لغة. حيث كان النظام يدعم تحويل الأرقام إلى اللغة العربية والإنجليزية فقط. وتم إضافة دوال لاختبار تفقيط الأرقام باللغات المدعومة مع إمكانية عرض اللغات المدعومة في النظام من خلال توفير API خاص بذلك.

---

## 2024-08-02
**Update Docs API and Support Param**

تحديثات في جزء التوثيق الخاص بالـ API، إضافة آلية للتعامل مع المتغيرات ضمن ملفات التوثيق حيث أصبح يتم جلب رابط الدومين الخاص بالـ API تلقائياً واستبداله بالـ localhost الذي كانت الأمثلة تعتمد عليه في التوثيق.
